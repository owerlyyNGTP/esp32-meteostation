# История изменений

Все значимые изменения в этом проекте будут записаны здесь.

## [1.0.1] - 2025-03-30

### Добавлено
- Разделение обновления времени и датчиков (таймеры)
- Глобальные переменные для хранения показаний
- Отдельная функция `updateDisplay()`
- Принудительный режим измерения BMP280 (forced mode)

### Изменено
- Оптимизирован вывод на OLED (убрано мерцание)

- — (ничего не исправлялось)

## [1.0.0] - 2025-03-28

### Добавлено
- Базовая версия: чтение AHT20 и BMP280
- Вывод на OLED 1.3"
- Подключение по Wi-Fi
- Синхронизация времени через NTP

### Известные проблемы
- Время и датчики обновляются с одинаковой частотой (5 секунд)
