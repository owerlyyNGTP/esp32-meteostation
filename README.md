# ESP32 MeteoStation v2.1.0

Автономная метеостанция на базе ESP32 с датчиками AHT20 (температура/влажность) и BMP280 (давление), выводом на OLED 1.3" (SH1106), отсылкой данных на сервер и синхронизацией времени через NTP.

## Возможности
- Чтение температуры, влажности, давления
- Вывод на OLED с обновлением времени каждую секунду (датчики — раз в 5 секунд)
- Синхронизация времени по Wi-Fi (NTP)
- Энергосберегающий режим опроса BMP280 (forced mode)
- Многопоточность (FreeRTOS): Опрос датчиков и экран на Core 1, отправка данных по сети на Core 0.
- Умное управление экраном: Активация по кнопке (GPIO 13), автозатемнение через 15 сек и сон через 20 сек.
- IoT-логика: Отправка данных в формате JSON на локальный Python-сервер (Flask) с проверкой API-токена.
- Локальная БД: Автоматическое ведение истории замеров в `meteo_data.csv`.


## Аппаратное обеспечение
- ESP32S NODEMCU-32
- Датчик AHT20 (температура/влажность)
- Датчик BMP280 (давление)
- OLED 1.3" (128x64, SH1106)

## Подключение
| Компонент           | ESP32   |
| :---                | :---    |
| SDA (датчики/экран) | GPIO 21 |
| SCL (датчики/экран) | GPIO 22 |
| VDD (датчики/экран) | 3.3V    |
| GND                 | GND     |
